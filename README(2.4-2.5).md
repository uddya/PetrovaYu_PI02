Вариант 2. Реализуйте рекурсивную функцию для вычисления степени числа (power(x, n)).
Работа алгоритма:
1. Объявляется функция def power(x, n).
2. Условие n==0 является базовым случаем рекурсии, то есть без него рекурсия будет вызывать себя бесконечно.
3. Условие n==1 является вторым базовым случаем рекурсии.
4. В остальных случаях, при n>1, возвращается значение x * power(x, n - 1).
Сложность: O(n).
Объяснение сложности: Функция делает ровно n рекурсивных вызовов (по одному на каждую единицу показателя степени). То есть время выполнения алгоритма пропорционально n.
(9) Чем отличаются перестановки от сочетаний?
Главное отличие перестановки и сочетания в том, что перестановка - это упорядоченный набор всех или части элементов множества, а сочетание - это неупорядоченный набор элементов.
